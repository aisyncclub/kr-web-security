# 1. 법적 의무

## 개인정보보호법 안전성확보조치 기준

### 법적 근거
- 개인정보 보호법 제29조, 시행령 제30조
- **고시 제2023-6호** (2023.9.22 시행, 2025.10.31 개정)

### 주요 조항 요약

| 조항 | 제목 | 핵심 요구사항 |
|------|------|-------------|
| 제4조 | 내부 관리계획 | 책임자 지정, 연 1회 이상 점검, 취급자 교육 |
| **제5조** | 접근 권한 관리 | 최소 권한 원칙, 개인별 계정, 변경 기록 3년 보관 |
| **제6조** | 접근통제 | 불법 접근 방지, 외부 접속 시 2FA(OTP/인증서) 적용 |
| **제7조** | 암호화 | 비밀번호 일방향 해시, 고유식별정보 AES-256, 송수신 TLS |
| **제8조** | 접속기록 | 1년(일반)/2년(5만명↑) 보관, 월 1회 점검 |
| 제9조 | 악성프로그램 방지 | 백신 설치 및 최신 업데이트 |

### 제7조 암호화 상세 (가장 중요)

```
필수 암호화 대상:
- 비밀번호 → 일방향 해시 (argon2id/bcrypt)
- 주민등록번호 → AES-256-GCM
- 여권번호, 운전면허번호, 외국인등록번호 → AES-256
- 신용카드번호, 계좌번호 → AES-256 (직접 저장 지양)
- 생체인식정보 → AES-256
- 송수신 → TLS 1.2 이상
```

### 소규모 사업자 특례
- 1만명 미만 정보주체 처리하는 소상공인: 내부 관리계획 **생략 가능**
- **단, 기술적 조치(암호화, 접근통제)는 면제 아님**

## 전자상거래법 보안 의무

### 거래기록 보존 (법 제6조, 시행령 제6조)

| 기록 유형 | 보존 기간 |
|-----------|-----------|
| 표시/광고 | 6개월 |
| 계약/청약철회 | **5년** |
| 대금결제/재화 공급 | **5년** |
| 소비자 불만/분쟁처리 | **3년** |

### 사업자 필수 표시 (법 제13조)
- 상호, 대표자, 사업장 소재지, 전화번호, 사업자등록번호

## 위반 시 제재

| 유형 | 금액 | 비고 |
|------|------|------|
| 과징금 | 전체 매출액 **3%** (2023.9 개정) | 반복 시 10% 추진 중 |
| 과태료 | 최대 **5,000만원** | 안전조치 위반 |
| 형사처벌 | 5년 이하 징역 / 5,000만원 이하 벌금 | 제71조 |
